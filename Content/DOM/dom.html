DOM

Das Document Object Model kurz DOM ist eine Programierschnitstelle für HTML und XML Dokumente.

Quelle: [05.02.2021] https://developer.mozilla.org/de/docs/Gecko-DOM-Referenz/Einf%C3%BChrung

Es bietet die Möglichkeit auf die verschachtelte Struktur des Elements zuzugreifen und so gezielt belibige teile zu verändern oder zu lesen. Dadurch ist es Scriptsprachen wie JavaScript möglich, mit dem durch HTML vom Webbrowser gerenderten inhalt zu Interagierren. Das bedeutet beispielsweise das gezielt auf der Webseite angezeigte Inhalte verändert werden können, ohne das eine ganze Seite neu geladen werden muss.

Die einzelnen DOM elemente sind in einer Baumstruktur angeordnet. Ein wichtiger Unterschied besteht dabei in der unterscheidung der Typen. Ein HTML Node kann beispielsweise auch ein Text oder ein Element sein. Oder aber auch einer nur ein Node. Stichwort: Vererbung

Quelle: [05.02.2021] https://javascript.info/basic-dom-node-properties

Mittels DOM kann auch auf die CSS stylesheets eines HTML Dokuments zugegriffen werden, auch wenn dieses in einer externen css Datei ausgelagert ist.

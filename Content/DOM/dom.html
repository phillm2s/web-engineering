DOM

Das Document Object Model kurz DOM ist eine Programierschnitstelle für HTML und XML Dokumente.

Quelle: [05.02.2021] https://developer.mozilla.org/de/docs/Gecko-DOM-Referenz/Einf%C3%BChrung

Es bietet die Möglichkeit auf die verschachtelte Struktur des Elements zuzugreifen und so gezielt belibige teile zu verändern oder zu lesen. Dadurch ist es Scriptsprachen wie JavaScript möglich, mit dem durch HTML vom Webbrowser gerenderten inhalt zu Interagierren. Das bedeutet beispielsweise das gezielt auf der Webseite angezeigte Inhalte verändert werden können, ohne das eine ganze Seite neu geladen werden muss.

Die einzelnen DOM elemente sind in einer Baumstruktur angeordnet. Ein wichtiger Unterschied besteht dabei in der unterscheidung der Typen. Ein HTML Node kann beispielsweise auch ein Text oder ein Element sein. Oder aber auch einfach nur ein Node. Stichwort: Vererbung

Quelle: [05.02.2021] https://javascript.info/basic-dom-node-properties

Mittels DOM kann auch auf die CSS stylesheets eines HTML Dokuments zugegriffen werden, auch wenn dieses in einer externen css Datei ausgelagert ist.

1. DOM Manipulation

Wie komme ich an das DOM Objekt?

Auf das gerenderte DOM kann mit dem Schlüssewort document zugegriffen werden. Dieses Objekt bietet wiederum eine Reihe von Methoden für den spezifischen Zugriff auf das gewünschte Element.

Die wichtigsten sind dabei:

Methode Beschreibung
getElementById(id) Gibt das Element mit der entsprechenden ID zurück.
getElementsByClassName(name) Gibt eine iterierbare HTMLCollection mit allen Elements die den übergebenen class Namen tragen zurück.
getElementsByTagName(tag) Gibt eine HTMLCollection mit allen Elements mit dem übergebenen tag zurück. Wird tag="*" Übergeben, werden alle Element Objekte des Dokuments zurückgegeben.

Diese Methoden waren nur solche, die direkt auf das document Objekt angewendet werden. Dazu kommt noch eine Vielzahl an Methoden um sich in der Verschachtelten Struktur des DOMs zu bewegen. Durch kind Elemente zu iterieren oder deren Attribute zu ändern.
